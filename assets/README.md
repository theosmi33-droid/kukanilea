# Branding Assets

This directory contains packaging and app icons for KUKANILEA.

Source reference used for the brand wordmark:

- https://register.dpma.de/DPMAregister/marke/register/3020232097482/DE

Generated assets:

- `icon.png` (master square icon, 1024x1024)
- `icon.icns` (macOS app icon)
- `icon.ico` (Windows app icon)
- `brand-wordmark.jpg` (original downloaded wordmark for traceability)
