# KUKANILEA – Benutzerhandbuch (Quickstart)

Willkommen bei KUKANILEA, Ihrem lokalen Betriebssystem für das Handwerk. Dieses Handbuch führt Sie durch die ersten Schritte.

## 1. Installation & Start
KUKANILEA ist eine **Offline-Anwendung**. Das bedeutet, alle Ihre Daten bleiben auf Ihrem Rechner.
- **macOS:** Öffnen Sie das DMG und ziehen Sie KUKANILEA in Ihren Programme-Ordner.
- **Windows:** Führen Sie die MSI-Datei aus.

Beim ersten Start führt das System eine **Hardware-Erkennung** durch, um die KI-Modelle optimal auf Ihren Rechner einzustellen.

## 2. Dokumente & OCR (PicoClaw)
Laden Sie Belege (Rechnungen, Lieferscheine) einfach per Drag-and-Drop hoch.
- Das System extrahiert automatisch Lieferanten, Beträge und Daten.
- Nutzen Sie den **Indigo-Vorschlag**, um KI-Ergebnisse zu prüfen.
- KUKANILEA **lernt** aus Ihren Korrekturen: Korrigieren Sie einen Wert einmal, merkt sich das System das Layout für die Zukunft.

## 3. Team-Zusammenarbeit (Mesh-Sync)
Sie können KUKANILEA im Team nutzen, ohne einen Cloud-Server zu benötigen.
- Öffnen Sie **Einstellungen > Team-Status**.
- Scannen Sie den **QR-Code** mit Ihrem Smartphone, um Belege direkt von der Baustelle hochzuladen.
- Andere Computer im gleichen WLAN finden sich automatisch und synchronisieren die Daten.

## 4. DATEV-Export
Am Monatsende können Sie alle Buchungsdaten für Ihren Steuerberater exportieren.
- Gehen Sie zu **Einstellungen > DATEV Export**.
- Wählen Sie den Monat und laden Sie das ZIP-Archiv herunter.

## 5. Datenschutz & Sicherheit
KUKANILEA nutzt einen integrierten **ClamAV-Virenscanner**. Jede hochgeladene Datei wird vor der Verarbeitung geprüft. Da keine Daten in die Cloud gesendet werden, sind Sie zu 100% DSGVO-konform.
