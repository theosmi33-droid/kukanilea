# KUKANILEA Beta is live

KUKANILEA `v1.0.0-beta.1` is available as a prerelease.

## What is included
- DMS with OCR, search, and tags.
- CRM, Leads, Tasks/Kanban.
- Workflows/automation.
- Local AI assistant (Ollama based).
- Local-first operation without cloud dependency.

## How to join
1. Clone the repository.
2. Run `./scripts/beta_quickstart.sh`.
3. Report feedback via GitHub Issues using the `beta` template.

- Runbook: `docs/runbooks/BETA.md`
