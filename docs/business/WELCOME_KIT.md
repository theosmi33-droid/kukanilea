# Willkommen in der digitalen Festung: KUKANILEA Enterprise
## Ihr persönliches Onboarding-Kit (v1.6.0)

Sehr geehrter KUKANILEA-Nutzer,

vielen Dank für Ihr Vertrauen in die lokale Intelligenz. Sie haben sich für 100% Datensouveränität entschieden. Hier sind die nächsten Schritte, um Ihre Enterprise-Edition scharfzuschalten.

### 1. Installation
1. Öffnen Sie die mitgelieferte `KUKANILEA.dmg`.
2. Ziehen Sie das KUKANILEA-Icon in Ihren `Programme`-Ordner.
3. Starten Sie die App über das Launchpad.

### 2. Aktivierung
Beim ersten Start erscheint der **Onboarding-Wizard**.
1. Kopieren Sie Ihren persönlichen **Activation Code** (Format: `KUK-XXXX-XXXX-XXXX-XXXX`).
2. Senden Sie diesen Code einfach per E-Mail an Ihren IT-Ansprechpartner oder direkt an `activation@kukanilea.de`.
3. Sie erhalten von uns eine `license.json` Datei zurück.

### 3. Lizenz-Import
1. Kopieren Sie die erhaltene `license.json` in folgendes Verzeichnis:
   *   **macOS:** `~/Library/Application Support/KUKANILEA/`
   *   **Windows:** `%APPDATA%\KUKANILEA`
2. Starten Sie die App neu.
3. Ihr System wechselt nun automatisch vom „Trial“ in den **Enterprise-Vollmodus**.

### 4. Support & Erste Hilfe
Sollten Sie Fragen zur Einrichtung haben, erreichen Sie unseren Enterprise-Support unter:
*   **Web:** [https://tophandwerk.de/support](https://tophandwerk.de/support)
*   **Diagnose:** Nutzen Sie den Button „System-Check“ im Bereich „Settings“, um ein Fehlerprotokoll für unsere Techniker zu erstellen.
