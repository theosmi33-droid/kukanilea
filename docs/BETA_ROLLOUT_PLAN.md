# BETA_ROLLOUT_PLAN

Date: 2026-02-21

## Scope
- macOS: primary supported beta platform.
- Windows: experimental while distribution signing evidence remains blocked.

## Rollout Window
- 2-4 weeks controlled pilot.

## Support Model
- Best-effort support for Windows during beta.
- Full support focus on macOS pilot users.

## Communication
- State platform support clearly in release notes.
- Link known blockers and gate status from `docs/RELEASE_GATES.md`.

## Feedback Channel
- GitHub Issues (preferred)
- Direct support contact (if defined by operations)

## Abort Criteria
- Confirmed security incident.
- Data-loss defect.
- More than 5 active P1 incidents without mitigation.

## Exit Criteria
- Beta KPIs and gate evidence reviewed.
- RC blockers reduced to explicit prerequisites only.
