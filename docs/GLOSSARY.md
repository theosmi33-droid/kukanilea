# Glossary

- ADR: Architecture Decision Record fuer dauerhaft relevante Architekturentscheidungen.
- Core-Freeze: Regeln, die unkontrollierte Stack-/Dependency-Aenderungen verhindern.
- DoD: Definition of Done mit pruefbaren Kriterien.
- ErrorEnvelope: Standardisiertes API-Fehlerformat.
- Offline-first: Kernfunktionen laufen ohne externe Cloud-Abhaengigkeit.
- PII: Personenbezogene Daten (z. B. Name, E-Mail, Telefon, IBAN).
- READ_ONLY: Betriebsmodus, in dem Mutationen gesperrt sind.
- Tenant: Isolierter Mandant mit strikt getrennten Daten.
- Triage: CI-/Runtime-Qualitaetsgate fuer Warnungen und Stabilitaet.
