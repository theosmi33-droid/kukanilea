# KUKANILEA Gold – Schnellstart-Guide für Pilotkunden

Herzlichen Glückwunsch zur Installation von KUKANILEA v1.5.0 Gold. Da wir höchste Ansprüche an den Datenschutz stellen, arbeitet das System zu 100% offline.

## 1. Aktivierung
Beim ersten Start von KUKANILEA wird ein Aktivierungs-Bildschirm angezeigt.

1. **Hardware-ID (HWID) kopieren:** Kopieren Sie den angezeigten Code (z.B. `eb8f7e5f...`).
2. **Schlüssel anfordern:** Senden Sie diesen Code an den Support (theosmi33@gmail.com).
3. **Lizenz einspielen:** Sie erhalten eine Datei `license.kukani`. Laden Sie diese im Aktivierungs-Fenster hoch oder kopieren Sie den Inhalt in das Textfeld.
4. **Vollbetrieb:** Nach dem Klick auf "Aktivieren" ist Ihr System dauerhaft und ohne Internetverbindung freigeschaltet.

## 2. Erste Schritte
- **PicoClaw Vision:** Gehen Sie auf den Vision-Tab und laden Sie ein Foto eines Typenschilds hoch. PicoClaw extrahiert in unter 500ms Hersteller, Modell und Seriennummer.
- **Wissensdatenbank:** Laden Sie PDF-Dokumente hoch. Das System indiziert diese lokal für die KI-Suche.
- **Sprachsteuerung:** Nutzen Sie das Mikrofon-Icon, um Aufgaben oder Kundennotizen per Stimme zu diktieren.

## 3. Updates
Sollte ein Update verfügbar sein, erscheint ein dezenter Hinweis im Dashboard. Da wir revisionssicher arbeiten, werden Updates nur installiert, wenn sie eine gültige RSA-Signatur besitzen.

---
**Sicherheitshinweis:** Ihre Daten verlassen niemals diesen Rechner. KUKANILEA besitzt keine Cloud-Anbindung.
