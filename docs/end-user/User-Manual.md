# KUKANILEA – Benutzerhandbuch (Entwurf)

Willkommen bei KUKANILEA, Ihrem souveränen Betriebssystem für das Handwerk.

## 1. Erste Schritte
### Installation
KUKANILEA wird in der Regel auf einer ZimaBlade Appliance in Ihrem lokalen Netzwerk installiert.
1. Schließen Sie die Appliance an Ihr Netzwerk an.
2. Rufen Sie `http://kukanilea.local` in Ihrem Browser auf.

### Anmeldung
Standard-Zugangsdaten (bitte nach dem ersten Login ändern):
- **Admin:** admin / admin
- **Entwickler:** dev / dev

## 2. Beleg-Zentrale (Upload)
Laden Sie Ihre Rechnungen und Dokumente einfach per Drag-and-Drop hoch.
- **Analyse:** Die lokale KI führt automatisch eine OCR-Erkennung durch.
- **Prüfung:** Korrigieren Sie erkannte Daten in der Review-Ansicht.
- **Ablage:** Dokumente werden GOBD-konform archiviert.

## 3. Zeiterfassung
Nutzen Sie den Timer, um Arbeitszeiten projektbezogen zu erfassen.
- **Start/Stop:** Erfassen Sie Echtzeit-Daten.
- **Export:** Laden Sie Wochenübersichten als CSV für die Buchhaltung.

## 4. CRM & Kontakte
Verwalten Sie Ihre Kunden und Lieferanten direkt lokal.
- **Suche:** Finden Sie alle Dokumente zu einem Kontakt über den KI-Assistenten.

## 5. KI-Assistent
Fragen Sie Ihren lokalen Assistenten nach Dokumenten oder Informationen.
- Beispiel: "Zeige mir alle Rechnungen von Hans Müller aus dem letzten Monat."
