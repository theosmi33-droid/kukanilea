# Metrics

## Reliability
- Chat success rate (target: 99% local).
- API error rate per endpoint.
- Request latency (p50/p95).

## Productivity
- Tasks closed per week.
- Time tracked per project.
- Search success rate and time-to-result.

## Ingestion
- Docs indexed per hour.
- Extraction latency.
- Index rebuild duration.

## Security
- Prompt-injection blocks per week.
- Policy denials by role.
- Cross-tenant access attempts (should be 0).
