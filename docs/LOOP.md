# Improvement Loop

## Weekly Loop
- Review metrics and customer feedback.
- Re-score backlog using WSJF.
- Plan a small, shippable slice with specs + tests.
- Release checklist and verify CI.

## Daily Loop
- Triage incidents and security findings.
- Update active spec/plan docs.
- Implement + test + document changes.
- Record decisions in ADRs.

## Release Checklist
- Specs, ADRs, and contracts updated.
- Tests green (ruff/black/pytest/smoke).
- Security checks (tenant isolation, injection corpus).
- Manual UX smoke for core flows.
