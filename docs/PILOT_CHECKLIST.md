# Checkliste für Pilotkunden

Bitte gehen Sie die folgenden Punkte durch, um die Stabilität und Funktionalität von KUKANILEA in Ihrer Betriebsumgebung zu testen.

## 1. Installation & Aktivierung
- [ ] Installer läuft fehlerfrei durch.
- [ ] Onboarding-Wizard zeigt die HWID an.
- [ ] Lizenzdatei `license.kukani` wird erfolgreich eingelesen.

## 2. Kernfunktionen
- [ ] **CRM:** Kunden anlegen und Dokumente zuordnen.
- [ ] **Aufgaben:** Erstellen und Verschieben von Aufgaben im Kanban-Board.
- [ ] **Wissen:** Erfolgreiche Suche in der lokalen Wissensdatenbank.
- [ ] **Vision:** Analyse eines Typenschilds via PicoClaw.
- [ ] **Voice:** Ausführen eines einfachen Sprachbefehls ("Erstelle Aufgabe...").

## 3. Performance & Stabilität
- [ ] Dashboard lädt in unter 1 Sekunde.
- [ ] Keine Abstürze bei I/O-intensiven Tasks (OCR).

## Feedback geben
Bitte senden Sie uns Ihre Beobachtungen, Verbesserungsvorschläge oder Fehlermeldungen an `kukanilea@gmail.com`.
Vielen Dank für Ihre Unterstützung!
