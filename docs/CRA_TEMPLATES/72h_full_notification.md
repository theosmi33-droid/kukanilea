# CRA 72h Full Notification Template
**Status:** Awareness + 72h
**To:** ENISA / National Cybersecurity Authority

## 1. Incident Details
- Detailed description of the vulnerability/incident.
- Technical indicators (CVE-ID if applicable).

## 2. Exploitation Evidence
- Evidence of active exploitation in the field.
- Number of affected tenants (if known).

## 3. Mitigation Plan
- Planned corrective measures.
- Estimated time to "Final Report" (14d limit).

## 4. Evidence Attachments
- Security scan reports.
- Log snapshots (PII-redacted).
